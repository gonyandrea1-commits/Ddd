# Tech Calculator OCR

Static web app for technician reports, stackable code tracking, OCR field capture, weekly/monthly earnings totals, and copy-ready MTPN notes.

## Files

- `index.html` — main app page
- `app.js` — app logic, OCR parsing, reports, storage, earnings export
- `baza.xml` — UI text, report templates, code prices, speed/eero options
- `styles.css` — app styling
- `manifest.json` — installable web app settings
- `sw.js` — service worker cache for GitHub Pages
- `icons/icon.svg` — app icon

## GitHub Pages

To publish:

1. Open repository settings.
2. Go to **Pages**.
3. Choose **Deploy from a branch**.
4. Select branch **main** and folder **/(root)**.
5. Save.

After GitHub builds it, the app should open at:

`https://gonyandrea1-commits.github.io/Ddd/`
